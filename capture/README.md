# Capture

Use `capture/` for quick, unprocessed intake.

Capture can be messy. It should not become permanent storage.

Every capture item should eventually become one of:

- deleted after confirmation if deletion is non-trivial
- moved to `sources/library/`
- summarized or synthesized into `wiki/`
- indexed as a source record
- moved to `assets/`
- moved to `system/migration/` if it is only migration bookkeeping
