# GB Wiki

This is a source-aware, LLM-maintained knowledge workspace.

It intentionally starts empty: no raw data, no copied source materials, and no imported history from another vault.

Core layout:

- `capture/` is for quick unprocessed intake.
- `sources/` is for source records, source maps, and selected source artifacts when they are useful.
- `wiki/` is for synthesized understanding.
- `system/` is for operating rules, templates, index, and log.
- `assets/` is for supporting media that is not itself source evidence.

Default workflow:

```text
capture -> triage -> ingest -> synthesize -> index/log -> lint
```

Read first:

1. `system/agent-instructions/source-philosophy.md`
2. `AGENTS.md`
3. `system/operating-model.md`
4. `system/index.md`
