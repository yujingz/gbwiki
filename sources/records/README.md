# Source Records

Use `sources/records/` for source records, ingestion records, and review records.

```text
sources/records/
  source-records/
  ingestion-records/
  review-records/
```

A source record points to an original source and records canonicality, owner, access method, relationship, ingestion mode, sensitivity, staleness risk, and related wiki/source notes.

A review record captures a source-quality review, second-model review, cleanup decision, lint result, or migration audit without rewriting source artifacts themselves.
