# Wiki

Do not use it as raw source storage.

Suggested lanes:

- `indexes/`
- `topics/`
- `people/`
- `organizations/`
- `projects/`
- `decisions/`
- `guides/`

Create a new wiki page only when the concept, entity, project, decision, or guide is durable enough to reuse.
